# smart_gladiator_task

Project Name: Task using codeIgniter

Technologies Used:
- PHP (CodeIgniter)
- MySQL
- HTML, CSS, Bootstrap 4
- jQuery (AJAX)

Features:
- User Registration (Signup)
- User Login (Signin)
- Email validation (no duplicate emails)
- Landing Page
- Real-time Chat (AJAX-based)


Database Setup:

1. Open phpMyAdmin
2. Create a new database (e.g., sg_task)
3. Import the file: database/database.sql
4. Update database config:

application/config/database.php

'hostname' => 'localhost',
'username' => 'root',
'password' => '',
'database' => 'sg_task',
